Fingertip

An experimental open source resolver for truly decentralized names.

Features

  • A lightweight, decentralized Handshake resolver
  • Zero-trust browser security using DANE/TLSA without any centralized certificate authorities
  • Access to external namespaces such as Ethereum Name Service
